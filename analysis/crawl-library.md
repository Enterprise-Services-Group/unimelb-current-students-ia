## What this means

### 1. The Library is the academic skills backbone

370 referencing style pages, 66 reference management pages, collection guides — the Library provides the academic infrastructure that underpins every student's research and writing. This is student-critical content.

### 2. Moderate hub connection — could be stronger

36 links to the student hub in 30 sampled pages is moderate. The hub links back to the library (221 contextual links from 13 domains). The connection exists but isn't prominent.

### 3. The referencing content is high-volume, high-value

370 pages on referencing styles is the largest single-content-section in any domain crawled. This is the kind of content students search for at 2am before an assignment deadline. It should be findable from the hub's academic skills section.

## Recommendations

1. **Add Library referencing to the hub's academic skills section.** A student on the hub looking for "how to reference" should find the Library's 370-page referencing guide.

2. **Cross-link the hub and library more prominently.** The library→hub connection (36 links in sample) and hub→library (221 contextual links from 13 domains) could both be stronger.

3. **No urgent issues.** The Library is well-maintained, well-structured, and serves a clear purpose. Low priority for structural fixes.
